# ☁️ Jay Cloud

Your Personal Cloud. Built From Scratch.

Jay Cloud is a self-hosted cloud platform developed by Jay Karan Chaturvedi.

## Features

- Portfolio Website
- Project Showcase
- Secure Admin Dashboard
- Application Hosting
- Cloud Storage
- Docker Management
- MongoDB Integration
- Monitoring
- CI/CD (Planned)

## Tech Stack

Frontend
- React
- Vite
- TypeScript
- Tailwind CSS

Backend
- Express
- MongoDB
- Dockerode

Infrastructure
- Docker
- Cloudflare Tunnel
- Ubuntu Server
