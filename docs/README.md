# Overview

Jay Cloud is a modern self-hosted cloud platform that combines application hosting, cloud storage, container management, monitoring, and portfolio management into one unified interface.

Instead of opening multiple tools like Portainer, MongoDB Compass, Uptime Kuma, and Cloudflare dashboards separately, Jay Cloud provides a single control center for managing the entire homelab.

The project is designed to grow into a complete developer platform while also serving as the public portfolio website for Jay Karan Chaturvedi.

---

# Vision

Jay Cloud aims to become a lightweight self-hosted alternative to platforms such as:

- Vercel
- Render
- Railway
- Google Drive
- Portainer

while remaining simple, developer-focused, and fully under the owner's control.

# Running Locally

Clone the repository

```bash
git clone https://github.com/Jaykaran24/jay-cloud.git
```

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm start
```
